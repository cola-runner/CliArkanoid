# CLI Arkanoid

A simple implementation of the classic Arkanoid game that runs in your terminal.

## How to Play

1.  **Run the game:**
    ```bash
    python arkanoid.py
    ```
2.  **Controls:**
    *   **Left Arrow Key:** Move the paddle left.
    *   **Right Arrow Key:** Move the paddle right.
    *   **'q' or ESC:** Quit the game.

3.  **Objective:**
    *   Use the paddle to bounce the ball and destroy all the bricks at the top of the screen.
    *   Don't let the ball fall below your paddle!
    *   You have 3 lives.
